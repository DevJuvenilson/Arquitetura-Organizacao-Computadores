Diagrama do Circuito
===================================================

DIAGRAMA SIMPLIFICADO:
----------------------

   Arduino Uno                   Protoboard
  ┌──────────────┐           ┌───────────────────┐
  │              │           │  + ● ← ─────────┐ │
  │  Pino 13 ●──┼───🔴──────┼──►               │ │
  │              │           │               [R 220Ω]
  │              │           │                  │ │
  │              │           │  Linha 10, col A ● │
  │              │           │                  │ │
  │              │           │              [LED +] (perna longa)
  │              │           │              [LED -] (perna curta)
  │              │           │                  │ │
  │              │           │  Linha 11, col A ● │
  │              │           │               ▲  │ │
  │   GND ●──────┼───⚫──────┼───────────────┘  │ │
  └──────────────┘           └───────────────────┘


TABELA DE CONEXÕES:
-------------------

 Origem           | Destino               | Cor do Fio
 ─────────────────┼───────────────────────┼───────────
 Arduino Pino 13  | Protoboard linha +    | Vermelho
 Protoboard linha+| Resistor 220Ω perna 1 | (interno)
 Resistor perna 2 | Linha 10, coluna A    | (interno)
 LED perna LONGA  | Linha 10, coluna E    | (interno)
 LED perna CURTA  | Linha 11, coluna E    | (interno)
 Linha 11, col A  | Arduino GND           | Preto


POSICIONAMENTO NA PROTOBOARD:
------------------------------

      a  b  c  d  e     f  g  h  i  j
   +  ●  ○  ○  ○  ○     ○  ○  ○  ○  ○   ← Linha positiva (borda)
   -  ○  ○  ○  ○  ○     ○  ○  ○  ○  ○   ← Linha negativa (borda)
   ...
   8  ○  ○  ○  ○  ○     ○  ○  ○  ○  ○
   9  ○  ○  ○  ○  ○     ○  ○  ○  ○  ○
  10  ●  ○  ○  ○  ●     ○  ○  ○  ○  ○   ← col A: Resistor | col E: LED (+)
  11  ●  ○  ○  ○  ●     ○  ○  ○  ○  ○   ← col A: GND      | col E: LED (-)
  12  ○  ○  ○  ○  ○     ○  ○  ○  ○  ○
   ...

  Legenda:
  ● = Furo ocupado
  ○ = Furo livre


ESQUEMA ELÉTRICO (simplificado):
----------------------------------

  5V (Pino 13)
       │
     [220Ω]     ← Resistor limita corrente (protege o LED)
       │
      [▶]       ← LED (triângulo = positivo para cima)
       │
      GND        ← Terra (referência 0V)


CÁLCULO DO RESISTOR:
---------------------
  Tensão Arduino:  5V
  Tensão LED:      ~2V (vermelho) / ~3V (azul/branco)
  Corrente ideal:  ~10-20mA

  Fórmula: R = (V_fonte - V_led) / I_led
  Exemplo: R = (5V - 2V) / 0.015A = 200Ω → usar 220Ω (mais próximo)
