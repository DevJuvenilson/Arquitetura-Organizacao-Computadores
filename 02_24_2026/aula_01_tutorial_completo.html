Identificar Componentes

🔍 Vamos Conhecer Cada Peça:

Arduino Uno

Arduino Uno R3 ┌─────────────────────────────┐ │ USB │ │ ┌───┐ │ │ │ │ ┌────┐ │ │ └───┘ │CHIP│ │ │ └────┘ │ │ ○ ○ ○ ○ ○ ○ ○ ○ ○ ○ ○ ○ │ ← Pinos Digitais (0-13) │ ○ ○ ○ ○ ○ ○ │ ← Pinos Analógicos (A0-A5) │ ○ ○ ○ │ ← GND, 5V, 3.3V └─────────────────────────────┘

Vamos usar: Pino 13 e GND

LED (Light Emitting Diode)

┌─┐ │ │ ← Cabeça transparente └─┘ │ Perna LONGA (+) ← ANODO (positivo) │ Perna CURTA (-) ← CATODO (negativo)
⚠️ SUPER IMPORTANTE:
LED TEM POLARIDADE! Não funciona se ligar ao contrário!
• Perna LONGA = Positivo (+)
• Perna CURTA = Negativo (-)

Resistor 220Ω

┌─────┐ ──┤█████├── └─────┘ Código de cores: 🔴 Vermelho - 🔴 Vermelho - 🟤 Marrom = 220Ω
💡 Por que usar resistor?
Arduino fornece 5V, LED aguenta 2-3V.
Resistor "come" o excesso, protegendo o LED!

Montar o Circuito - Parte 1: LED

🔧 Montagem PASSO A PASSO:

3.1 - Colocar LED na Protoboard

Protoboard (visão de cima):

a b c d e f g h i j ... 10 ○ ○ ○ ○ ● ○ ○ ○ ○ ○ ← LED perna LONGA aqui 11 ○ ○ ○ ○ ● ○ ○ ○ ○ ○ ← LED perna CURTA aqui ...
  1. Pegar LED
  2. Identificar perna LONGA (olhe com atenção!)
  3. Inserir perna LONGA na linha 10, coluna E
  4. Inserir perna CURTA na linha 11, coluna E
  5. Pressionar gentilmente até firmar
⚠️ Cuidado: Não force! Pernas do LED são frágeis e podem dobrar.
💡 Dica Visual: Perna longa geralmente fica do lado ACHATADO do LED.

📸 Checkpoint: Como deve estar agora

  • LED firmemente inserido na protoboard
  • Perna longa na linha 10
  • Perna curta na linha 11
  • LED na vertical (não torto)

    Montar o Circuito - Parte 2: Resistor

    🔧 Adicionar Resistor:

    4.1 - Preparar Resistor

    1. Pegar resistor 220Ω (vermelho-vermelho-marrom)
    2. Dobrar as pernas em formato "L"
    3. Deixar cerca de 1cm de cada lado

    4.2 - Colocar Resistor

    + + ●... ← Uma perna do resistor aqui (linha +) ... 10 ● ○ ○ ○ ● ← LED perna longa + outra perna resistor 11 ○ ○ ○ ○ ● ← LED perna curta
    1. Uma perna na linha 10, coluna A (mesma linha do LED+)
    2. Outra perna na linha + (positivo da borda)
    💡 Por que mesma linha?
    Na protoboard, furos da mesma linha estão conectados internamente!
    Então resistor e LED+ ficam conectados automaticamente.

    📸 Checkpoint

  • Resistor firmemente inserido
  • Uma perna na linha 10, coluna A
  • Outra perna na linha + (borda)
  • Resistor fazendo "ponte"
    • 5

    Montar o Circuito - Parte 3: Jumpers

    🔧 Conectar ao Arduino:

    5.1 - Jumper Vermelho (Positivo)

    1. Pegar jumper vermelho
    2. Conectar uma ponta no Arduino pino 13
    3. Conectar outra ponta na linha + da protoboard
    Arduino Protoboard ┌──────────┐ ┌──────────┐ │ 13 ●───┼─────🔴────┼→ + ● │ │ │ │ │

    5.2 - Jumper Preto (Negativo/GND)

    1. Pegar jumper preto
    2. Conectar uma ponta no Arduino GND
    3. Conectar outra ponta na linha 11, coluna A (LED-)
    Arduino Protoboard ┌──────────┐ ┌──────────┐ │ │ │ 10 ●─┐ │ │ │ │ [LED] │ GND ●───┼─────⚫────┼→ 11 ●─┘ │ └──────────┘ └──────────┘
    💡 Cores dos Jumpers:
    Vermelho = Positivo (+) / Alimentação
    Preto = Negativo (-) / GND (terra)
    Não é obrigatório, mas ajuda MUITO a organizar!

    🎉 Circuito Completo!

    Diagrama Final:

    Arduino Uno Protoboard ┌──────────┐ ┌──────────────┐ │ 13 ●───┼───🔴──────┼→ + ●─┐ │ │ │ │ │ │ │ │ │ [R220Ω] │ │ │ │ │ │ │ │ │ 10 ●─┤ │ │ │ │ [LED] │ │ │ │ 11 ●─┘ │ │ GND ●───┼───⚫──────┼→ ● │ └──────────┘ └──────────────┘

    ✅ Checklist Final do Circuito

  • Jumper vermelho: Pino 13 → Linha +
  • Resistor: Linha + → Linha 10 coluna A
  • LED perna longa: Linha 10 coluna E
  • LED perna curta: Linha 11 coluna E
  • Jumper preto: GND → Linha 11 coluna A
  • Nenhum fio solto ou tocando outro
    • 🛑 ANTES DE CONTINUAR:
    Verifique CADA conexão novamente!
    Um erro agora pode queimar seu LED!

    Escrever o Código - Parte 1: Setup

    📝 Entendendo a Estrutura do Arduino:

    Todo programa Arduino tem 2 funções principais:

    ┌─────────────────────────────┐ │ setup() │ │ Roda UMA VEZ ao iniciar │ │ → Configura pinos, serial │ ├─────────────────────────────┤ │ loop() │ │ Roda INFINITAMENTE │ │ → Lógica principal aqui │ └─────────────────────────────┘

    ✏️ Apagar o código atual e digitar:

    📄 led_controller.ino — Parte 1: Definições e Setup
    // =================================================== // ISW044 - Aula 01: LED Controller // Pisca um LED no pino 13 // =================================================== // Definição da constante do pino const int LED_PIN = 13; // Pino onde o LED está conectado // =================================================== // SETUP: Roda uma vez ao ligar o Arduino // =================================================== void setup() { // Configura o pino 13 como SAÍDA (OUTPUT) pinMode(LED_PIN, OUTPUT); // Inicia a comunicação serial (para debug) Serial.begin(9600); Serial.println("LED Controller iniciado!"); }
    💡 O que cada linha faz?
    const int LED_PIN = 13; → Cria uma constante chamada LED_PIN com valor 13
    pinMode(LED_PIN, OUTPUT); → Diz ao Arduino que o pino 13 vai ENVIAR sinal
    Serial.begin(9600); → Abre canal de comunicação com o computador
  • Código apagado (tela limpa)
  • Variável LED_PIN declarada
  • Função setup() digitada corretamente
  • pinMode() configurado como OUTPUT

    Escrever o Código - Parte 2: Loop

    📝 Adicionar a função loop():

    Logo abaixo do setup(), adicione o loop:

    📄 led_controller.ino — Parte 2: Loop Principal
    // =================================================== // LOOP: Roda infinitamente // =================================================== void loop() { // LIGAR o LED digitalWrite(LED_PIN, HIGH); // HIGH = 5V = LED aceso Serial.println("LED: LIGADO"); delay(1000); // Espera 1 segundo (1000ms) // DESLIGAR o LED digitalWrite(LED_PIN, LOW); // LOW = 0V = LED apagado Serial.println("LED: DESLIGADO"); delay(1000); // Espera 1 segundo }
    💡 O que cada linha faz?
    digitalWrite(LED_PIN, HIGH) → Coloca 5V no pino → LED acende
    digitalWrite(LED_PIN, LOW) → Coloca 0V no pino → LED apaga
    delay(1000) → Pausa de 1000 milissegundos = 1 segundo

    📄 Código Completo (para conferir):

    📄 led_controller.ino — Código Completo
    // ISW044 - Aula 01: LED Controller const int LED_PIN = 13; void setup() { pinMode(LED_PIN, OUTPUT); Serial.begin(9600); Serial.println("LED Controller iniciado!"); } void loop() { digitalWrite(LED_PIN, HIGH); Serial.println("LED: LIGADO"); delay(1000); digitalWrite(LED_PIN, LOW); Serial.println("LED: DESLIGADO"); delay(1000); }
  • Função loop() adicionada
  • digitalWrite(HIGH) para ligar
  • digitalWrite(LOW) para desligar
  • delay(1000) nos dois estados
  • Código completo conferido

    Carregar o Código no Arduino

    🚀 Enviando para o Arduino (Upload):

    1. Certifique-se que o Arduino está conectado
    2. Clique no botão → (Carregar/Upload)
    3. Ou pressione Ctrl + U
    4. Observe as luzes RX/TX piscando no Arduino
    5. Aguarde a mensagem de conclusão

    O que acontece durante o upload:

    1. IDE compila o código novamente 2. Envia via USB para o Arduino 3. Luzes TX/RX piscam rapidamente 4. Arduino reinicia automaticamente 5. Seu programa começa a rodar! Mensagem esperada: ✅ Carregamento concluído.
    ⚠️ Erro "Port not found" ou "avrdude: error"?
    • Verifique se a porta COM correta está selecionada
    • Tente desconectar e reconectar o Arduino
    • Feche o Monitor Serial antes de fazer upload
    💡 As luzes TX/RX piscando significam que dados estão
    sendo transferidos entre o computador e o Arduino. Normal!
  • Cliquei em Carregar (Ctrl+U)

    Desafio: Modificar o Código

    🏆 Hora de Experimentar!

    Agora que o básico funciona, tente esses desafios:

    🔴 Desafio 1 — Mudar a velocidade:

    Altere o delay para 500ms e veja o LED piscar mais rápido!

    delay(500); // Antes era 1000 (1 segundo) — agora 0,5 segundo

    🟡 Desafio 2 — LED mais tempo ligado:

    Deixe o LED aceso por 2 segundos e apagado por 0,5 segundo:

    digitalWrite(LED_PIN, HIGH); delay(2000); // 2 segundos ligado digitalWrite(LED_PIN, LOW); delay(500); // 0,5 segundo apagado

    🟢 Desafio 3 — Código Morse SOS:

    Faça o LED piscar o sinal SOS (... --- ...):

    // Função auxiliar para piscar void piscar(int duracao) { digitalWrite(LED_PIN, HIGH); delay(duracao); digitalWrite(LED_PIN, LOW); delay(200); } void loop() { // S = 3 pontos curtos piscar(200); piscar(200); piscar(200); delay(400); // O = 3 traços longos piscar(600); piscar(600); piscar(600); delay(400); // S = 3 pontos curtos piscar(200); piscar(200); piscar(200); delay(2000); // Pausa entre repetições }
    💡 Lembre-se: Após modificar o código, sempre:
    1. Verificar (Ctrl+R)
    2. Carregar (Ctrl+U)
    3. Observar o resultado!
